# sony-av-indicator

## Configuration

Adjust the IP address of your AVR:

    TCP_IP = "xxx.xxx.xxx.xxx"

Adjust the port of your AVR:

* Port 8080 (STR-DA1800, STR-DN1030)
* Port 50001 (STR-DN1060)
* Port 33335 (All Others)

    TCP_PORT = 33335

## Usage

    ./sony-av-indicator.py

## System requirements

* Ubuntu 16.04
* Sony STR-DN-860
